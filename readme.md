## 🖥️ Preview

> SwayFX + Waybar + Kitty + Starship
> Clean tiling, blur effects, and a fast terminal workflow.

---

## ⚙️ Dependencies

Make sure you have these installed before using the config:

### 🧠 Core Shell

* `zsh`
* `starship`

### 🔧 CLI Tools

* `eza`
* `bat`
* `ripgrep`
* `fd`
* `fzf`

### 🔌 Zsh Plugins

* `zsh-autosuggestions`
* `zsh-syntax-highlighting`

---

### 🖥️ Terminal / Editor

* `kitty`
* `neovim`

---

### 🧊 Wayland / WM

* `swayfx`
* `waybar`
* `wofi` *(⚠️ use version `1.5.2-1` — newer versions break due to CSS issues)*

---

### 🔗 System Integration

* `xdg-desktop-portal`
* `xdg-desktop-portal-wlr`
* `polkit-gnome`

---

### ⌨️ Input / IME

* `fcitx5`

---

### 🔊 Audio

* `pavucontrol`

---

### 📸 Utilities

* `grim` (screenshots)
* `slurp` (region selection)
* `wl-clipboard` (clipboard)

---

### 🎨 Fonts

* `nerd-fonts`

### 🧪 Notes
* This setup assumes Arch Linux
* Uses SwayFX, not vanilla Sway (blur, shadows, rounded corners)
* Some configs are hardware-specific (tablet mapping, outputs, etc.)
* Newer versions of wofi crash due to a css error. Use `1.5.2-1`
